# Scan Hints

Copy this file to `scan_hints.md` and customize it with your own knowledge.
This content is appended to the Claude scan prompt to improve extraction accuracy.

## Process

Common values: Washed, Natural, Honey, Anaerobic, Wet Hulled, Semi-Washed.
Swedish labels may say "Tvättad" (Washed) or "Naturell" (Natural).

## Bean Type

Common species: Arabica, Robusta, Liberica.
Common varieties: Bourbon, Typica, Gesha/Geisha, SL28, SL34, Caturra, Catuai, Catimor, Castillo, Pacamara, Heirloom.

## Country Grown

Common origins: Ethiopia, Colombia, Brazil, Kenya, Guatemala, Costa Rica, Honduras, Peru, Rwanda, Burundi, Indonesia, India, Mexico, El Salvador, Panama, Yemen.
If multiple countries are listed it is likely a blend.

## Roast Level

Common values: Light, Medium-Light, Medium, Medium-Dark, Dark.
Swedish labels may say "Ljus" (Light), "Mellanbrygg" (Medium), or "Mörk" (Dark).
Nordic roasters typically roast lighter.
